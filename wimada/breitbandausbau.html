1. Breitbandausbau Thema Auswählen

1.1
Um das Thema Breitbandausbau zu aktivieren öffnen Sie in der Themenleiste (rechter Bilschirmrand) die Rubrik "Breitbandausbau" durch einen Klick mit der linken Maustaste auf das + Symbol. Es öffnen sich die Unterrubriken "Ausschreibung", "Phase1", "Phase2", "Phase3", die Sie wiedrum mit einem Klick der linken Maustaste auf das + Symbol öffnen können.

Setzen Sie nun im gewünschten Thema ein Aktivierungshäkchen um das Thema anzuzeigen und es gleichzeitig abfragebereit zumachen.

2. Breitbandausbau Ausbaugebiet Suchen

2.1
Um sich ein Ausbaugebiet anzeigen zulassen, wählen sie im linken Menübereich den Reiter "Themensuche" und dort den Unterpunkt "Ausbaugebiete_bfp". Es öffnet sich die Layersuche in der Sie Sucheinstellungen vornehmen können.

2.2
Wählen Sie nun einen "Operator" und einen "Wert" nachdem Sie suchen wollen

2.3
Klicken sie anschließend auf "Suchen"
Zur nicht exakten Suche verwenden sie den Operator "ähnlich" und den Platzhalter %.
Für Datumsangaben verwenden sie bitte das Format TT.MM.JJJJ.

2.4
Stellen Sie sicher das die "Spaltenweise Darstellung" aktiviert ist (wie im Bsp. dargestellt) Ihr Suchergebnis wird in Tabellenform angezeigt.
Sie können Sich Ihr suchergebnis nun auch in der Karte anzeigen lassen. Wählen Sie hierfür das gewünschte Gebiet aus indem Sie es mit einen Häkchen aktivieren und anschießend auf den Button
"in der Karte anzeigen lassen" klicken

3. Breitbandausbau Statistik

3.1
Um sich die Statistik eines Ausbaugebietes anzeigen zu lassen wählen Sie im linken Menübereich den Reiter "Themensuche" und dort den Unterpunkt "Ausbaugebiete_bfp". Es öffnet sich die Layersuche.

3.2
Um sich die Statistk eines bestimmten Amtes anzeigen zulassen, klicken Sie nun auf "Suche Räumlich Eingrenzen..."

3.3
Es öffnet sich ein Kartenfenster in dem Sie nun das entsprechende Gebiet eingrenzen können. Dafür stellen Sie bei "Geometrie übernehmen von" den Layer "Amtsbereiche MV" ein. Danach klicken Sie in der Werkzeugleiste auf den Button "vorhandende Geometrie hinzufügen". Wählen Sie nun, mit einem Klick in der Karte, den gewünschten Amtsbereich aus.

3.4
Durch das Scrollen nach Unten erreichen Sie den "Suchen" Button. Bevor Sie auf "Suchen" klicken, tragen Sie im darüber liegendem Feld "Anzahl Treffer anzeigen:" eine 9999 ein. (Die Suche kann einen Moment in Anspruch nehemn)

3.5
Stellen Sie sicher das die "Spaltenweise Darstellung" aktiviert ist (wie im Bsp. dargestellt)
Scrollen Sie bis ans ende der Tabelle und klicken SIe auf das Summenzeichen

3.6
Nun werden Ihnen die Summen, Durchschnittswerte, Mindest- und die Höchstwerte der jeweiligen Spalte berechnet.
